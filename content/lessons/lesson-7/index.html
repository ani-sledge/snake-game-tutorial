To change the direction of the snake, we need to change the direction of it's first segment. Add the following function to snake.js:

var changeDirection = function(direction) {
  snake[0].direction = direction;
}

Next, we'll have CHUNK call changeDirection when an arrow key is pressed. Add the following line at the end of snake.js:

  CHUNK.onArrowKey(changeDirection);

Syntax Breakdown

`CHUNK.onArrowKey` Tells chunk to call the changeDirection function every time an arrow key is pressed.
